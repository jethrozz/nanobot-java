# 工具使用指南

## 内置工具

### read_file
读取文件内容
- 参数：path (文件路径)
- 用途：查看文件内容

### write_file
写入文件内容
- 参数：path (文件路径), content (文件内容)
- 用途：创建或修改文件

### exec
执行 Shell 命令
- 参数：command (要执行的命令)
- 用途：运行系统命令
- 注意：有超时限制和安全检查

### message
发送消息到指定频道
- 参数：channelId (频道ID), content (消息内容)
- 用途：向其他聊天平台发送消息

## 工具使用建议
1. 读取文件前先确认路径存在
2. 写入文件时注意备份重要数据
3. 执行命令时确保安全性
4. 发送消息时确认频道正确
